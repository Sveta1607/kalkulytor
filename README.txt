========================================
  Калькулятор стоимости услуг
  Инструкция по запуску
========================================

1. Откройте папку проекта:
   «калькулятор стоимости услуг»

2. Запуск (любой способ):

   Вариант А — двойной клик по файлу:
   • Откройте index.html двойным щелчком.
   Страница откроется в браузере по умолчанию.

   Вариант Б — через браузер:
   • Откройте браузер (Chrome, Edge, Firefox и т.д.).
   • Меню «Файл» → «Открыть файл» (или Ctrl+O).
   • Выберите файл index.html в папке проекта.

   Вариант В — из редактора/терминала:
   • В VS Code / Cursor: правый клик по index.html → «Open with Live Server»
     (если установлено расширение Live Server).
   • Или в терминале из папки проекта:
     npx --yes serve .
     затем откройте в браузере адрес, который покажет serve (обычно http://localhost:3000).

3. Как пользоваться:
   • Выберите тип услуги (Клининг / Ремонт / Фриланс).
   • Введите объём: для клининга — площадь в м², для остальных — часы.
   • При необходимости отметьте «Срочный заказ» и/или «Выезд».
   • Нажмите «Рассчитать цену» — итог появится ниже.

4. Изменение цен:
   Откройте файл calc.js. В начале файла объект PRICES и константы
   URGENT_PERCENT, VISIT_FEE — меняйте числа под свои тарифы.
